# Yulu-bike-demand-analysis

## Objective
Analyse electric bike demand pattern using python and Tableau to identify key trends and business insights

## Tools
Python(Pandas, Matplotlib, Seaborn, Scipy)

## Project Structure
- data/: Dataset
- notebooks/: Python analysis
- dashboard/: Tableau visuals & video

## key Insights
- Evening = 345 rides, the highest across all time segments.
- Demand rises sharply after 4 PM and peaks around 6â€“7 PM.
- Indicates office commute + leisure rides.
- Leisure usage on weekends is strong.
- Demand is not limited to office commuters.
- Demand drops when too cold or too windy.
